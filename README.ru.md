# Stevens Shop - Интернет-магазин с Telegram ботом

Учебный проект интернет-магазина на чистом PHP с использованием MySQL и интеграцией с Telegram ботом.

## Технологии

- Backend: PHP 7.4+
- База данных: MySQL
- Расширение: MySQLi
- Бот: Python 3.14, pyTelegramBotAPI
- Frontend: HTML5, CSS3, JavaScript
- Безопасность: Prepared Statements, сессии, защита от XSS

## Архитектурные решения
- Модульная структура - разделение на переиспользуемые компоненты (header, footer)
- MySQLi с Prepared Statements - защита от SQL-инъекций
- Сессии PHP - управление состоянием пользователя и корзиной
- Нормализация базы данных - отдельные таблицы для товаров, категорий, брендов, корзины
- htmlspecialchars() - защита от XSS-атак
- Telegram Bot API - интеграция с внешним сервисом для запросов товаров

## Реализованные возможности
### Система авторизации
- Регистрация пользователей с валидацией
- Вход/выход из системы
- Аутентификация на основе сессий
- Защита функционала корзины только для авторизованных пользователей

### Каталог товаров
- Динамическая генерация каталога из базы данных
- Полнотекстовый поиск по названию товара
- Фильтрация по категориям
- Фильтрация по брендам
- Индикатор наличия товара (в наличии / под заказ)
- Отображение изображений товаров

### Страница товара
- Детальная информация о товаре (название, описание, цена, изображение)
- Таблица характеристик товара
- Информация о бренде и категории
- Отображение количества на складе
- Функция добавления в корзину

### Корзина покупок
- Добавление товаров в корзину
- Просмотр корзины со всеми товарами
- Удаление товаров из корзины
- Сохранение корзины в базе данных (для каждого пользователя)
- Расчёт итоговой суммы в реальном времени
- Оформление заказа с модальным окном успеха

### Интеграция с Telegram ботом
- Просмотр всех товаров (/products)
- Поиск товаров по названию (/search)
- Получение случайного товара (/random)
- Проверка наличия товара (/check)
- Интерактивная навигация через клавиатуру
- Прямое подключение к MySQL из бота
- Получение данных о товарах в реальном времени

### Структура базы данных
- Таблица пользователей с аутентификацией
- Таблица товаров с ценами и остатками
- Категории для организации товаров
- Бренды для классификации товаров
- Таблица корзины, связывающая пользователей с товарами и количеством
